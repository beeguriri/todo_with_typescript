# todo_with_typescript
타입스크립트와 뷰로 만드는 todo


## 프로젝트 생성
```bash
#뷰설치
$ npm install vue
$ npm install -g @vue/cli c
$ npm i -g @vue/cli-init

#버전 확인
$ vue -V

#설치 안될때
#powershell 관리자관한 실행
$ Set-ExecutionPolicy Unrestricted

#프로젝트 생성
$ vue create vue-todo
```
